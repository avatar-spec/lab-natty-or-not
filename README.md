# Gerando um poema utilizando IA

## 📒 Descrição
Utilizando o chatgpt gerei um poema sobre liberdade 

## 🤖 Tecnologias Utilizadas
Chatgpt

## 🧐 Processo de Criação
Eu enviei o seguinte input para gerar o meu poema: "Gere pra mim um poema sobre liberdade que seja o mais natural possível"

## 🚀 Resultados

Liberdade é o vento que passa sem avisar,  
é o canto do pássaro que não tem hora de voltar.  
É o rio que segue seu curso sem fim,  
sem medo do que o futuro pode ser para si.

Liberdade é o olhar que se perde no horizonte,  
sem pressa, sem algemas, sem querer um norte.  
É o passo que avança sem pensar no caminho,  
é a escolha que se faz sem peso, sem espinho.

Liberdade é o campo onde a flor se abre,  
sem pedir licença, sem esperar aplausos.  
É o sol que brilha, a chuva que cai,  
sem perguntar se é certo ou errado, se faz bem ou faz mal.

Liberdade é simples, mas é imensa,  
não precisa de explicação, apenas de presença.  
É estar aqui, e ser quem somos,  
sem medo de ser, sem querer ser outros.
